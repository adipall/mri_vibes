complex_analysis_check.m is the file that compares diagonalization
done in matlab to that done in Salinas with sa_eigen. It also checks for a reduced
FRF.

You will want to keep all these files together in the same
directory.
complex_analysis_check.m
cmfrf.m
cmfrf_SA.m
dfrf.m
dfrf_tilde.m
dggev_evect.m
dggev_ev.m
diagonal_test.m
manoj.m
mfrf.m
neweigvec.m
simran.m

In the complex_analysis_check.m file you will want to cd to the
directory where you ran sa_eigen
